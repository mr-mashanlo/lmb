# Minimalist WordPress Theme

**Minimalist WordPress Theme** is a custom WordPress theme focused on simplicity, clean design, and ease of use.  
It is built for those who value minimalism and want to create a modern website without unnecessary clutter.

## Features
- Minimalist design with no distractions  
- Lightweight and fast performance  
- Fully responsive layout for all devices  
- Built with **PHP** and standard WordPress features  

## Technologies
- **WordPress** (theme)  
- **PHP** (core logic)  
- HTML, CSS (markup and styling)
